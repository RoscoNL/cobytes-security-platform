# Cobytes Security Platform

Een professioneel security scanning platform gebouwd voor Cobytes, geïntegreerd met Pentest Tools API voor uitgebreide beveiligingsscans.

## 🚀 Features

- **Multi-scan support**: Poort scans, vulnerability scans, SSL checks, DNS lookups
- **Real-time dashboards**: Live monitoring van scan resultaten
- **API Integration**: Pentest Tools API voor professionele scanning
- **Cross-platform**: Werkt op Windows, macOS en Linux
- **Modern UI**: Responsive design met intuïtieve interface

## 🛠️ Tech Stack

- **Backend**: Node.js + Express
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Database**: MariaDB (toekomstige integratie)
- **API**: Pentest Tools RESTful API

## 📋 Prerequisites

- Node.js 14+ 
- npm 6+
- Git
- Pentest Tools API key
